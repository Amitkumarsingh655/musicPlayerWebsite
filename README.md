# Antigravity Music Player

A visually stunning, premium dark-mode music player built with React and Vite. It features responsive layouts, glassmorphism design techniques, smooth animations, and a custom single-page application (SPA) router.

## Getting Started

To run the application locally, ensure you have Node.js installed and configured.

1. Install the dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Technologies Used

- **React 18**: The core frontend framework.
- **Vite**: Ultra-fast build tool and development server.
- **Lucide React**: For highly consistent, modern iconography.
- **CSS3 Variables & Backdrop Filters**: Providing a premium deep-dark aesthetic with dynamic glassmorphism and subtle gradient overlays.
